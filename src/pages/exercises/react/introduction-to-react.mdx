## Coding Challenges

### 1. Create a Simple React Component

Create a functional component called `Greeting` that takes a `name` prop and renders "Hello, [name]!".

<CodingChallenge
  initialCode={`
function Greeting(props) {
  // Your code here
}

// Render the component
render(<Greeting name="React Learner" />);
`}
  solution={`
function Greeting(props) {
return <h1>Hello, {props.name}!</h1>;
}

// Render the component
render(<Greeting name="React Learner" />);
`}
/>

### 2. Implement a Toggle Button

Create a functional component called `ToggleButton` that toggles between "ON" and "OFF" when clicked.

<CodingChallenge
  initialCode={`
function ToggleButton() {
  // Your code here
}

// Render the component
render(<ToggleButton />);
`}
  solution={`
function ToggleButton() {
const [isOn, setIsOn] = React.useState(false);

return (

<button onClick={() => setIsOn(!isOn)}>{isOn ? "ON" : "OFF"}</button>
); }

// Render the component
render(<ToggleButton />);
`}
/>
