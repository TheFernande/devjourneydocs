## Coding Challenges

### 1. Create a Functional Component

Create a functional component called `Welcome` that takes a `name` prop and renders a welcome message.

<CodingChallenge
  initialCode={`
function Welcome() {
  // Your code here
}

// Render the component
render(<Welcome name="React Learner" />);
`}
  solution={`
function Welcome({ name }) {
return <h1>Welcome, {name}! We're glad you're here.</h1>;
}

// Render the component
render(<Welcome name="React Learner" />);
`}
/>

### 2. Component with Multiple Props

Create a `UserCard` component that accepts `name`, `age`, and `occupation` as props and displays them in a structured format.

<CodingChallenge
  initialCode={`
function UserCard() {
  // Your code here
}

// Render the component
render(<UserCard name="Alice Johnson" age={28} occupation="Software Developer" />);
`}
  solution={`
function UserCard({ name, age, occupation }) {
return (
<div className="user-card">
<h2>{name}</h2>
<p>Age: {age}</p>
<p>Occupation: {occupation}</p>
</div>
);
}

// Render the component
render(<UserCard name="Alice Johnson" age={28} occupation="Software Developer" />);
`}
/>

### 3. Component Composition

Create a `ParentComponent` that renders two instances of a `ChildComponent`. The `ChildComponent` should accept a `message` prop.

<CodingChallenge
  initialCode={`
function ChildComponent() {
  // Your code here
}

function ParentComponent() {
// Your code here
}

// Render the ParentComponent
render(<ParentComponent />);
`}
  solution={`
function ChildComponent({ message }) {
return <p>{message}</p>;
}

function ParentComponent() {
return (
<div>
<h1>Parent Component</h1>
<ChildComponent message="I'm the first child" />
<ChildComponent message="I'm the second child" />
</div>
);
}

// Render the ParentComponent
render(<ParentComponent />);
`}
/>

## Further Practice

To deepen your understanding of React Components:

1. Create a component that uses conditional rendering based on props.
2. Build a list component that renders an array of items passed as a prop.
3. Experiment with creating a reusable button component that accepts custom styles and click handlers as props.

Remember, components are the building blocks of React applications. Practice creating and composing components to build your React skills!
